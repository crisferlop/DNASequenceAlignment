char* readFile ( char *  filePath)

Esta funcion lee un archivo y retorna los caracteres leidos, la cadena de caracteres esta dispuesta en memoria dinamica por lo que es responsabilidad del programador limpiar la memoria de este.

Parámetros
filePathes la direccion de el archivo representada mediante una cadena de caracteres
Devuelve
un puntero nulo si el archivo no existe, si no retorna el contenido
short int writeOnFile ( char *  filePath,
char *  contents,
short int  overwrite 
)

Funcion encargada de escribir sobre un archivo.

Parámetros
filePathes la direccion del archivo en el que se escribira
contentses el contenido a escribir en el archivo
overwritesi el valor es 1 la cadena de caracteres sera el que contenga el archivo, en otras palabras sobrescribir el contenido del archivo con la cadena de caracteres, si es 0 entonces agregara al contenido del archivo la cadena de caracteres deseada.
Devuelve
0 en el caso de que no se pueda escribir en el archivo. 1 si fue exitoso
